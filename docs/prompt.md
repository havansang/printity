# AI Prompt Context

This project is a T-shirt design editor.

Tech stack:
- React
- Fabric.js v7
- Vite

Main features:

- Canvas editor
- Layer management
- Multi surface design
- Auto save

Goal:

Build a production ready design editor similar to Printify.

Constraints:

Use React hooks.
Use Fabric.js 7 API.
Avoid global variables.
Use modular components.
